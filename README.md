# Dictionnaire
Ajouter vos mots dans le dictionnaire avec vos equivalent 
exemple : 
 - chelou -> bizarre
 - bizarre -> chelou

## Getting Started

```
cd DicoWesh
make
./dicowesh
```

### Prerequisites

GCC
